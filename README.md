<h1 align="center">Optional Class</h1>

<p align="justify">
    Welcome to the <b>Optional Class</b> repository! This repository provides hands-on demos explaining the need for
    <b>Optional</b> in Java to handle null values safely, preventing <b>NullPointerException</b>. It covers key methods
    for creating, retrieving, and manipulating <b>Optional</b> objects, helping you write cleaner and more robust Java
    code. </p>
</p>

<hr>

<h3 align="left">üìå Table of Contents</h3>
<ol align="left">
    <li>Identify need of Optional Class</li>
    <li>Methods to retrieve value from Optional Value: get(), orElse(), orElseGet(), orElseThrow()</li>
    <li>Methods to check if value exists in Optional Value: isEmpty(), isPresent(), ifPresent(),
        ifPresentOrElse()</li>
    <li>Methods to generate Optional Value: of(), ofNullable(), empty()</li>
    <li>Methods to manipulate Optional Value: filter(), map(), flatMap(), or(), stream()</li>
</ol>
